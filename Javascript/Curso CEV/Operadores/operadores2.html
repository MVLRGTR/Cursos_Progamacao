Operadores 2

Operadores Relacionais

  • 5 > 2 -> true
  • 7 < 4 -> false
  • 8 >= 8 -> true
  • 9 <= 7 -> false
  • 5 == 5 -> true
  • 4 != 4 -> false

Operadores de Identidade

  • 5 == 5 true
  • 5 == '5' true //Nesse caso estamos analisando somente o valor e não o tipo
  • 5 === '5' false //Nesse caso estamos analisando o valor e o tipo ao mesmo tempo

Operadores Lógicos

  • ! -> Negação
  • && -> conjunção(significa e em lógica)
  • || -> disjunção(significa ou em lógica)

Operador Ternário

Sintaxe

teste ? True(se a condição for verdadeira) : False (Se a condição for falsa)
